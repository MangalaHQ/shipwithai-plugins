# shipwithai-plugins

**SDLC Toolkit** — Monorepo chứa 30 Claude Code plugins covering full Software Development Life Cycle.

> Core IP của ShipWithAI. Mỗi plugin giải quyết 1 pain point cụ thể trong SDLC.

---

## Quick Start

```bash
# Clone repo
git clone git@github.com:ShipWithAI/shipwithai-plugins.git
cd shipwithai-plugins

# Install dependencies
npm install

# Build all plugins
npm run build

# Test all plugins
npm run test

# Build specific plugin
npm run build -- --filter=planning-assistant
```

## Development Guide

### Create a New Plugin

```bash
./scripts/create-plugin.sh my-new-plugin
```

This scaffolds: `plugins/my-new-plugin/` with package.json, README, src/, tests/.

### Plugin Structure

```
plugins/my-plugin/
├── package.json         # name: @shipwithai/my-plugin
├── README.md            # User-facing documentation
├── CHANGELOG.md         # Auto-generated by changesets
├── src/
│   ├── index.ts         # Entry point
│   ├── commands/        # Plugin commands
│   └── utils/           # Plugin-specific utils
└── tests/
    └── index.test.ts
```

### Code Review Process

```
Author → PR → Leonard (delivery check) → Link (quality gate, CODEOWNERS) → Merge
```

Contributors: See [CONTRIBUTING.md](./CONTRIBUTING.md)

## Publish Model (Hybrid)

```
Develop: 1 monorepo (Turborepo)
    ↓ changesets → version bump
Publish: Each plugin independently
    ↓ GitHub Actions
Users: Install individual plugins
```
